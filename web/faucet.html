Where is the full frontend?

The gamified earn hub (games, quizzes, shrine visit) is specified for koto.isekai-pool.com (Laravel + Livewire) — same infrastructure family as the pool — and is not shipped as static HTML in this repo.

This site (isekai-pool.com) documents the public JSON API you can call from any app once the operator enables it.

API (backend)

Base URL: https://api.isekai-pool.com/api/v1/faucet — /balance, /status, /recent, POST /claim (Cloudflare Turnstile + wallet).
