Pooled mining

Community Koto stratum pool: koto.mofumofu.me — algorithm yescryptR8G, username {your KOTO address}.{worker name}. Typical settings: ~0.5% pool fee, min payout 0.1 KOTO, PROP (confirm on the pool site).

Stratum ports (difficulty tiers): 3301 mobile · 3302 middle · 3303 high · 3304 cloud

./cpuminer --algo=yescryptR8G -o stratum+tcp://koto.mofumofu.me:3302 -u YOUR_KOTO_ADDRESS.worker1 -p x

Legacy minerd-style example from the pool: minerd -a yescryptR8G -o stratum+tcp://koto.mofumofu.me:3302 -u Address.Worker

FAQ

            

                
Pool vs solo?
For regular payouts, use stratum at koto.mofumofu.me with yescryptR8G. Solo uses RPC against a full node (see above).

                
What algorithm flag?
Pools document yescryptR8G (e.g. -a yescryptR8G / --algo=yescryptR8G). Run --help on your miner build if a variant name differs.

                
Other coins?
Yenten and Tidecoin are also on this hub.
