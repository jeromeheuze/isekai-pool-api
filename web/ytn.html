Mining setup

            
Install dependencies, compile cpuminer-opt (ARM64 or x86_64), then run the miner. Full copy-paste steps with tabs are on the mining guide.

            
    
                
  • Install build tools and libraries (see guide).
    • 
                
  • Clone cpuminer-opt and compile.
    • 
                
  • Generate a YTN receiving address in your wallet.
    • 
                
  • Run cpuminer with this page’s Quick connect command.
    • 
                
  • Watch for accepted in the log.

FAQ

            

                
What hardware do I need?
Any CPU works; Pis and desktops are common. See the hashrate table above for ballpark numbers.

                
Is this free?
Using our HTTP API for status and RPC proxy is free. Mining rewards come from the network when you find blocks.

                
What algorithm is YTN?
YespowerR16 — tuned for CPU mining.

                
Solo vs pool?
This page describes solo-style mining against an RPC node. Pool mining would use a different URL and stratum.
