# Isekai Adventure — Game Design Document
**Engine: Godot 4 · Payment: GameGlass.Live · Currency: KOTO / YTN**
**Platform: Web (HTML5) + Desktop · Publisher: 10k Game Studio**

---

## Elevator Pitch

A free top-down RPG set in a fantasy world where CPU mining powers magic. Play the entire game without ever touching crypto. Mine KOTO in the real world and the game notices — sending you item drops, achievement badges, and Steam-style event rewards. Want that exclusive yokai companion? Spend KOTO via GameGlass. Want nothing to do with crypto? Also fine. The game is complete either way.

**The Fortnite model. Except the V-Bucks are earned by mining a Japanese privacy coin.**

---

## The Big Picture

Isekai Adventure is simultaneously:
1. **A standalone game** — fun without any mining or crypto involvement
2. **The flagship demo for GameGlass.Live** — proves CPU coin payments work in games
3. **The deep engagement layer of isekai-pool.com** — the faucet games are the tutorial, this is the main game
4. **A hub for the Japan Empire Network** — quests send players to partner sites for real-world answers
5. **A showcase for 10k Game Studio** — easter eggs reference every game in the catalog

## Setting & Lore

The world of **Isekailand** runs on **Cryptomana** — magical energy that can only be generated by CPU-based proof-of-work. ASICs don't work here. Only the heat of real CPUs creates magic.

Five kingdoms, each themed around a real CPU coin:

| Kingdom | Coin | Aesthetic | Algorithm |
|---------|------|-----------|-----------|
| **Kotojima** | KOTO | Feudal Japan, torii gates, yokai, privacy magic | yescryptR8G |
| **Yentenshire** | YTN | Fantasy Europe, windmills, gold coins | yespowerR16 |
| **Tidehaven** | TDC | Coastal fishing village, tides, sea magic | yespower-b2b |
| **Sugarland** | Sugarchain | Pastel candy world, sweet magic | yespower |
| **Monerowood** | XMR | Dark forest, shadows, invisible paths | RandomX |

The player starts in **Kotojima** — the most developed kingdom, the home base of isekai-pool.com.

## Core Gameplay Loop

```
Arrive in Isekailand (tutorial — no crypto required)
    ↓
Explore, complete quests, solve puzzles — pure game
    ↓
[Optional] Link your mining wallet in settings
    ↓
Real mining triggers Steam-style event rewards (item drops, badges)
    ↓
[Optional] Spend KOTO on cosmetics via GameGlass
    ↓
Endgame: rebuild the broken CPU shrine and restore Cryptomana to the world
```

**The golden rule: every single step marked [Optional] is genuinely optional.**  
The game is fully completable, fully enjoyable, without a wallet, without mining, without spending a single KOTO.

## Mining Milestone Rewards (Optional Layer)

Players who link their mining wallet get Steam-event style rewards — tracked server-side via the pool API. No blockchain state in the game. No transactions mid-gameplay. Just: mine → game notices → reward arrives.

### How It Works
1. Player enters KOTO wallet address in Settings → Mining Rewards (optional, skippable)
2. Game polls `api.isekai-pool.com/api/v1/faucet/status?wallet=k1xxx` periodically
3. Server detects milestones → queues item drop notification
4. Next time player opens game: "You earned a reward while mining!"
5. Item stored in save data — no blockchain, no mid-game wallet interaction

### Mining Milestones

| Milestone | Reward | Trigger |
|-----------|--------|---------|
| Link your wallet | Welcome chest (random item) | One-time |
| Mine for 1 hour | "Novice Miner" badge | Pool API uptime |
| Mine for 24h straight | Rare Torii Fragment item | Pool API uptime |
| Find your first block | "Block Finder" title + rare item | Block detection |
| 7-day mining streak | Exclusive "Devoted Miner" cosmetic | Streak in Redis |
| Find 10 blocks total | Legendary "Hash Sage" weapon skin | Cumulative count |

### Block Found — Special Moment
When your miner finds a real KOTO block:
- In-game notification: *"Your CPU shrine pulses — a block has been found in Kotojima!"*
- Random rare item drops into inventory
- "Block Found Charm" cosmetic unlocked permanently
- Rarest, most meaningful reward in the game

### Design Notes
- All mining rewards are cosmetic — zero gameplay advantage
- Non-miners never feel blocked — mining panel hidden under Settings
- No crypto pop-ups during normal gameplay
- Game never requires a wallet to play, progress, or complete

## Spending KOTO — GameGlass.Live Integration

KOTO is an **optional premium currency**. Think V-Bucks or Gems — cosmetic unlocks, extra story content, vanity items. The game never locks gameplay behind KOTO. You earn KOTO by mining — or you simply never engage with it at all.

### What KOTO Buys (cosmetics + extras only)

| Item | Price | Type |
|------|-------|------|
| Yokai Companion pet | 3 KOTO | Cosmetic — follows player |
| Premium Gear Set | 2 KOTO | Visual skin — no stats |
| Extra Quest Line | 2 KOTO | Bonus story content |
| Supporter Badge | 1 KOTO | In-game title |
| Name in Credits | 10 KOTO | Permanent credit |
| Rare Dye Pack | 1 KOTO | Character color customization |

### What KOTO Never Buys
- Gameplay advantages
- Required story content
- Access to kingdoms (all kingdoms unlockable through play)
- Better stats, damage, speed

### GameGlass Implementation (Godot 4)

```gdscript
# Only triggered when player explicitly opens the shop
func open_shop_item(item_name: String, amount_koto: float):
    var widget_url = "https://gameglass.live/widget"
    widget_url += "?coin=KOTO&amount=" + str(amount_koto)
    widget_url += "&wallet=" + PlayerData.wallet_address
    widget_url += "&item=" + item_name.uri_encode()
    
    $GameGlassOverlay.load_url(widget_url)
    $GameGlassOverlay.show()

func _on_payment_confirmed(item_name: String):
    $GameGlassOverlay.hide()
    unlock_cosmetic(item_name)
    show_notification("✨ " + item_name + " unlocked!")
```

Shop is accessed via the Yokai Market in Koto Village — player must walk in voluntarily. Never pushed mid-gameplay.

## Real Mining Integration (Optional, Background)

The game checks your mining status silently in the background — only if you've linked a wallet. Players who never link a wallet have zero different gameplay experience.

```gdscript
# Runs every 5 minutes in background — silent, non-blocking
func _check_mining_rewards():
    if PlayerData.wallet_address == "":
        return  # not linked, skip entirely
    
    var http = HTTPRequest.new()
    add_child(http)
    http.request("https://api.isekai-pool.com/api/v1/faucet/status?wallet=" 
        + PlayerData.wallet_address)
    
    # On response: check for new milestones, queue item drops
    # Player sees notification only on next scene load — never mid-combat
```

**Key principle:** Mining integration is invisible to non-miners. It runs in the background and surfaces rewards gently — a notification on the main menu, never an intrusion.

## Why This Works

- **For players**: Free game, fun first, crypto is invisible unless you want it
- **For miners**: Their CPU work gets meaning — item drops, badges, a world that notices them
- **For people who hate crypto games**: They never see a wallet prompt, never feel pushed
- **For isekai-pool.com**: Game is the best possible marketing — miners discover the pool through play
- **For GameGlass.Live**: First clean real-world integration — cosmetics only, no P2W
- **For KOTO**: Western on-ramp, first real spending use case outside exchanges
- **For 10k Game Studio**: Flagship that markets every other game via easter eggs
- **For the Japan Empire Network**: Cross-site quest layer drives real traffic

**What makes it different from every failed crypto game:**
Every crypto game failed because the economy became the game. Here the economy is a quiet background layer. The game is the game. KOTO is just a nice bonus for the people who want it.
